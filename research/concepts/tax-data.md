# Tax-Data

**Institution**: Policy Simulation Library (PSL)
**Type**: Open-source microdata preparation package
**Language**: Python

## What It Is

Tax-Data prepares representative microdata files for use with Tax-Calculator. It combines IRS Public Use File data with Current Population Survey data to create synthetic tax records that approximate the actual distribution of US taxpayers.

## Significance

The unglamorous but critical data layer. Without representative microdata, microsimulation can calculate individual scenarios but can't estimate aggregate or distributional effects. Tax-Data solves this by creating statistically representative synthetic populations.
