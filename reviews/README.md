# Reviews

Review rounds organized by timestamp.

## Folder naming

Format: `YYYY-MM-DDTHHMM` (ISO 8601 basic format, UTC)

Examples:
- `2024-12-17T0858` = December 17, 2024 at 08:58 UTC
- `2024-12-21T2342` = December 21, 2024 at 23:42 UTC

## Contents

Each folder contains:
- Review documents (from external reviewers or simulated via academic-referee plugin)
- `*-synthesis.md` - Actionable summary of feedback for revisions
